# OOP_python
 
Mini-projects in Python for practicing object-oriented programming (OOP).
 
Each project lives in its own folder and includes the code plus a separate README describing it.
 
## Projects
 
- [game_character_project](./game_character_project) — a game character class: health, level, inventory.
- [order_project](./order_project) — an order class: add/remove items, calculate total, find the most expensive item.
## Stack
- Python 3
