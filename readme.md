# OOP_python
 
Projects in Python for practicing object-oriented programming (OOP).
Each project lives in its own folder and includes the code plus a separate README describing it
 
## Featured Project
 
- [electronics_shop_project](./electronics_shop) — a full warehouse management system for an electronics store. Built around 4 interacting classes (`Product`, `Inventory`, `Sale`, `StoreSystem`) that handle stock management, sales processing, and reporting through a console menu. Includes an automated test suite.
## Mini-Projects
 
- [game_character_project](./game_charact) — a game character class with methods: health, level, inventory.
- [Cafe_order_project](./cafe_order) — an order class with methods: add/remove items, calculate total, find the most expensive item.
- [Airport_system_project](./airport_system) — a flight class with methods: add/remove passengers, check for duplicate passports, count adults and children, check flight capacity
