# OOP_python

Mini-projects in Python for practicing object-oriented programming (OOP).
Each project lives in its own folder and includes the code plus a separate README describing it.

## Projects
- [game_character_project](./game_charact) — a game character class: health, level, inventory.
- [Cafe_order_project](./cafe_order) — an order class: add/remove items, calculate total, find the most expensive item.
- [Airport_system_project](./airport_system) — a flight class: add/remove passengers, check for duplicate passports, count adults and children, check flight capacity.

## Stack
- Python 3
